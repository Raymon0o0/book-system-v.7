О базе данных

Приложение использует MySQL для хранения информации о книгах, пользователях и заявках.

MySQL 8.0+

Параметры подключения

JDBC URL
jdbc:mysql://localhost:3306/librarydb
Имя пользователя
root
Пароль
********

Быстрые действия

К списку книг Добавить книгу

Примеры SQL-запросов

SELECT * FROM books;
-- Показать все книги
SELECT COUNT(*) FROM books;
-- Количество книг
Важная информация
  • Данные сохраняются постоянно (MySQL — дисковая база данных).
  • Для доступа к базе данных используйте MySQL Workbench, phpMyAdmin или командную строку.
  • При перезапуске приложения данные не теряются.
  • Перед развёртыванием убедитесь, что база данных создана и доступна.
